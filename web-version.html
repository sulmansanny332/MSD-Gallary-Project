🌐 MSD Web Version

MSD Shop Management System (Web Version) is a browser-based POS system built using HTML, CSS, JS, PHP, and MySQL. It runs on a local server and provides full business control through a dynamic dashboard.

⚙️ Core Working Process

• Real-time Dashboard (Profit, Investment, Revenue, Stock)
• Product Management (Add/Edit/Delete/Restore)
• Sales System with automatic stock deduction
• Investment module linked with stock updates
• Smart Expiry Alert System (30–60 days warning)
• Bank slip & Electricity bill management
• Recycle bin for safe recovery (30 days)

🧾 POS & Billing System

✔ Fully functional POS system
✔ Automatic bill calculation
✔ Stock auto-update after sale
✔ Supports Thermal Printing (55mm & 80mm)
✔ Fast receipt generation using browser print

🧰 Tech Stack

HTML CSS JavaScript PHP MySQL Local Server Thermal Print 55mm/80mm
