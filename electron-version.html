💻 MSD Electron Version

MSD Shop Management System (Electron App) is a fully offline desktop solution built for Windows and Linux using Electron.js and SQLite.

⚙️ Core Features

• Real-time Dashboard (Profit, Investment, Revenue, Stock)
• Product Management (Add/Edit/Delete/Restore)
• Sales System with automatic stock deduction
• Investment module linked with inventory
• Smart Expiry Alert System (30–60 days warning)
• Bank slips & Electricity bill records
• Recycle bin with 30-day recovery

🧾 Desktop POS System

✔ Fully Offline System (No Internet Required)
✔ Fast Performance
✔ SQLite Local Database
✔ Supports Thermal Printing (55mm & 80mm)
✔ Direct receipt printing from app
